# 🚴 Sales Analytics Dashboard (Excel)

## 📊 Project Overview
This project presents an interactive **Sales Analytics Dashboard** built using Microsoft Excel to analyze sales performance, customer behavior, and product profitability. The dashboard enables users to explore key business metrics such as revenue, profit, discounts, and regional performance through dynamic visualizations.

---

## 🎯 Project Objectives

1. Compare **sales and profit (%) across different months** to identify seasonal trends and performance variations.

2. Analyze and compare **sales generated by different customers** to identify high-value customers.

3. Evaluate **sales contribution (%) across different countries** to understand regional performance.

4. Compare **units sold (%) by different countries** to analyze demand distribution geographically.

5. Analyze **profit trends over different months across two years** to identify growth patterns and fluctuations.

6. Examine the **total sales distribution across different customer segments** to understand segment-wise contribution.

7. Provide dynamic insights by displaying:
   - Total Sales  
   - Total Profit  
   - Top Selling Product ID  
   - Total Units Sold  
   based on selected slicers for interactive analysis.

---

## 📁 Dataset Description

The dataset contains transactional sales data with the following fields:

- **Customer Details**: `Customer_ID`, `Customer_Name`, `Country`, `Segment`  
- **Product Details**: `Product_ID`, `Product_Name`, `Manufacturing Price`, `Sale Price`  
- **Sales Metrics**:  
  - Units Sold  
  - Gross Sales  
  - Discounts  
  - Net Sales  
  - COGS (Cost of Goods Sold)  
  - Profit  
- **Time Information**: `Date`, `Month`, `Quarter`, `Year`  
- **Discount Band**: Categorization of discount levels

## 🔍 Key Insights

- High discounts increase sales volume but may reduce profit margins  
- Certain customer segments contribute more to revenue  
- A few products generate the majority of profits (Pareto effect)  
- Seasonal trends are visible in monthly and quarterly sales  
- Regional differences impact overall performance

## 🚀 How to Use

1. Open the Excel file  
2. Navigate to the **Dashboard Sheet**  
3. Use slicers to filter data by:
   - Segment  
   - Country  
   - Product  
   - Time period  
4. Analyze charts and insights dynamically
